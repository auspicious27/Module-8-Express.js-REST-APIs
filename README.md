# Module 8 - Express.js and REST APIs Notes

This folder contains classroom-ready notes for Module 8.

## Files

- `index.html` - Complete formatted notes in Simple Hinglish.
- `student-management-api/` - Runnable Student Management REST API project.

## Topics Covered

- Express Framework
- Routing and Middleware
- REST API Principles
- CRUD Operations
- Input Validation
- Error Handling
- MVC Architecture
- Project 8: RESTful API Project
- CRUD APIs
- Validation and error handling
- API testing

Open `index.html` in a browser to teach the module.

## Run Student Management API

```bash
cd student-management-api
npm install
node server.js
```

Then open:

- `http://localhost:3000/`
- `http://localhost:3000/api/students`
